# Latencyfix
Makes every players on a Minecraft server takes the same knockback (regardless of their connections)

### Reminder for the custom knockback
- Player on the ground:
  - horizontal: 0.42
  - vertical: 0.3
- Player in the air:
  - horizontal: 0.42
  - vertical: 0.3
- Player on sprint:
  - horizontal: 2
  - vertical: 1.3
  - yaw pitch: 0.5
