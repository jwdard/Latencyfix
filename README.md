# Latencyfix
Makes every players on a Minecraft server takes the same knockback (regardless of their connections)

### Reminder for the custom knockback
- Ground:
  - horizontal: 0.42
  - vertical: 0.3
- Air:
  - horizontal: 0.42
  - vertical: 0.3
- Sprint:
  - horizontal: 2
  - vertical: 1.3
  - yaw pitch: 0.5
